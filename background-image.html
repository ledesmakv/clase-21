En general, se prefiere usar el tag desde el HTML.
1. Accesibilidad: permite que usuarios que usan un lector de imagenes sepan que alli hay una imagen, y lean su contenido mediante la etiqueta "alt"
2. SEO: Los motores de busqueda encuentran estas imagenes, y aparecen si el usuario las busca.
3. Velocidad de carga: El CSS debe cargar entero (incluidas las imagenes) antes de que la web empiece siquiera a mostrarse. En cambio, si la imagen esta en el HTML, la web se carga y solo despues comienzan a cargarse las imagenes.
